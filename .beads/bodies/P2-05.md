# P2-05 — Phase 2→3 gate detection + banner

## Summary
When score reaches 150, transition from Phase 2 to Phase 3: set `state.phase = 3`, set `state.phaseName = 'PHASE 3: AMBUSH'`, re-baseline tick rate, trigger a 2-second phase-transition banner. Uses the same `transitionToPhase()` helper introduced by P1-04.

Per PRD §3.1 thresholds; §6.5 gate; §12 gates table.
