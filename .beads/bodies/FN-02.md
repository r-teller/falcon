# FN-02 — Game loop and tick clock (pausable)

## Summary
Implement the core game-loop module: `requestAnimationFrame`-driven render loop with a fixed-tick accumulator that decouples render rate from game-tick rate (per PRD §18 — "fixed-tick accumulator"). Tick rate is a state variable (cells/sec) consumable by other modules. A `paused` flag (toggled by FN-07) halts tick advancement without tearing down the render loop, so overlays still draw.

## Persona
Primary: System (engine core).
Secondary: End User (responsive movement and pause are user-perceived properties of this loop).

## Changes Needed
| File | Action | What Changes |
|------|--------|-------------|
| `snake_opus.html` | Edit | Add `state` object with `tickRate` (initial 6 cells/sec per PRD §3.16), `lastFrameTs`, `tickAccumulator`, `paused` (bool). Add `loop(ts)` function using `requestAnimationFrame` that computes `dt = ts - lastFrameTs`, accumulates into `tickAccumulator`, and calls a `tick()` hook (no-op until FN-04+) every `1000 / tickRate` ms while not paused. Render always runs (so paused overlay stays visible). Start with `requestAnimationFrame(loop)` once DOM is ready. |

## Acceptance Criteria
### System
- [ ] `state.tickRate` is mutable; changing it changes the tick frequency on the next frame.
- [ ] When `state.paused === true`, the `tick()` callback is NOT invoked but the render frame still runs (verify with a `console.log` probe inside `tick()` during a temporary pause).
- [ ] Browser tab visibility loss (which can spike `dt`) does not cause a flood of catch-up ticks. Cap accumulated dt at e.g. 250 ms per frame to avoid spiral-of-death.

### End User
- [ ] On a 60 Hz monitor with `tickRate = 6`, the `tick()` hook fires ~6 times per second (verifiable via a temporary frame-counter log).

## Patterns to Reuse
- Standard `requestAnimationFrame` + fixed-tick accumulator pattern. Reference: PRD §18 row "Game loop".

## Testing Strategy
- Manual: temporary `console.log` inside `tick()`; open devtools, observe tick rate matches `state.tickRate`. Toggle `state.paused = true` from the console, verify ticks stop.
- Regression: FN-01 scaffold still renders.
