# FN-07 — HOLD POSITION overlay + P/Esc toggle

## Summary
Implement pause toggle: P or Esc flips `state.paused`. When paused, show a "HOLD POSITION" overlay (centered text, dimmed background). Per PRD §3.22, pause must freeze the tick clock AND all world-event timers (Ambush, wormhole countdown, bonus expirations) — but the world-event timers are wired in later phases. FN-07 establishes the `state.paused` flag and the overlay; FIN-04 wires the timer-freezing for Ambush/wormhole/bonuses.

Per PRD §3.22 (pause behavior); §4.3 (controls P/Esc).

## Persona
Primary: End User (pause is a user-controlled mechanic).

## Changes Needed
| File | Action | What Changes |
|------|--------|-------------|
| `snake_opus.html` | Edit | Extend FN-03 key handler: when key is `KeyP` or `Escape`, flip `state.paused`. Add HUD overlay div `#hud-pause` with text "HOLD POSITION", centered, hidden by default. In `renderHud()` (FN-06), toggle the overlay's visibility on `state.paused`. The tick clock freeze is already provided by FN-02 (it skips `tick()` when paused). |

## Acceptance Criteria
### End User
- [ ] Pressing P while running shows the "HOLD POSITION" overlay and stops patrol movement.
- [ ] Pressing P again resumes (overlay hides, patrol resumes movement, score and game state preserved).
- [ ] Esc has the same effect as P.
- [ ] Overlay is visually distinct (centered text, dimmed background using `rgba(0,0,0,0.5)` or similar).
- [ ] Pause does NOT lose any state (score, patrol length, position).

### System
- [ ] `state.paused` toggles correctly; pause/resume are idempotent (P-press while paused only resumes once).
- [ ] World-event timer freezing for Ambush / wormhole / bonus is OUT of scope here and tracked separately as FIN-04.

## Patterns to Reuse
- Single DOM overlay with display toggle: `pauseEl.style.display = state.paused ? 'flex' : 'none'`.

## Testing Strategy
- Manual: press P during gameplay, verify movement halts and overlay appears. Press P again, verify resume. Repeat with Esc.
- Regression: FN-02 loop, FN-03 input, FN-04 patrol movement still work.
